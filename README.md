# VKmon Frontend

Одностраничное приложение на React + Vite, которое повторяет основные сценарии бэкенда VKmon: авторизация, работа с VK-аккаунтами и группами, настройки автопостинга, AI-генерация и мониторинг логов/тарифов.

## Быстрый старт
1. Установите зависимости:
```bash
npm install
```
2. Создайте файл переменных окружения (опционально):
```bash
cp .env.example .env.local
```
3. Запустите приложение:
```bash
npm run dev
```
По умолчанию фронтенд стартует на `http://localhost:5173` и ходит в API `http://localhost:8080`.

## Конфигурация
- `VITE_API_BASE_URL` — базовый URL бэкенда VKmon. Если не задан, используется `http://localhost:8080`.
- Все API вызовы описаны в `src/api/client.ts`. При недоступности сервера используются заглушки из `src/api/mockData.ts` для демонстрации интерфейса.

## Карта экранов
- **Дашборд** — быстрый обзор подключенных аккаунтов, групп и возможностей API.
- **Авторизация** — формы `/login` и `/register` на базе DTO `Login` и `Register`.
- **VK аккаунты** — список, добавление аккаунта (`CreateVkAccountRequest`) и просмотр групп (`VkGroupDto`).
- **Автопостинг** — отправка настроек расписания (`ScheduleDto`) и привязка к группе/профилю.
- **AI контент** — форма генерации поста через `/ai/post` c параметрами `AiGenerateRequest`.
- **Логи и биллинг** — вывод логов `/logs` и справочник тарифов (UUID планов используются в регистрации).

## Технологии
- React 18 + TypeScript
- Vite 5
- React Router для маршрутизации
- TanStack Query для работы с API и кешем
- Axios для HTTP-клиента
